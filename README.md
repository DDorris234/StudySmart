# StudyBuddy (C++ Console App)

StudyBuddy is a simple C++ console application that takes user input notes and generates a basic summary and simplified version of the text.

## Features
- Takes user input through console
- Extracts first sentence as a summary
- Converts text to lowercase for simplified output
- Demonstrates basic C++ functions and string manipulation

## How to Run
1. Open project in Visual Studio
2. Build and run using Local Windows Debugger
3. Enter notes when prompted

- C++ functions
- String manipulation
